## How to Run Locally
1. Clone the repo https://github.com/Niranjan1310/Virtual-Try-On/edit/main/README.md
   ```bash
   git clone 
   cd React-Virtual-Try-on
2. Run backend
   ```bash
   python -m venv venv
   venv\Scripts\activate  # on Windows
   pip install -r requirements.txt
   python main.py

3. Run frontend
   ```bash
   cd frontend
   npm install
   npm start
